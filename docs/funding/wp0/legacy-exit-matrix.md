## Mandatory exit query contract

Before deleting any legacy reconciler, `FinanceOperationsOwner` must produce a
fresh report grouped by frozen `adapter_version` and normalized status, list
every non-terminal row by internal ID in a restricted artifact, and prove its
provider/chain disposition. A zero count from age alone is insufficient; refund
and ambiguous-broadcast states must be terminally resolved.
